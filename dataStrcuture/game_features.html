Games Features

In this module, we will define the data structure required to extract games features.

Games

These features are computed at the game level and contains: + metadata: general information about game date, team identity, competition, … + stadium information: coordinates, ssurface, capacity, … + condition day time, attendane, meteo, … + dominace: The head to head record between the home and away team

The document also contains tagret information i.e, the different prediction targets such as the match-outcome, score, goal-difference,… They are identified with the tgt_ prefix.
